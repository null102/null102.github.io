一.

首先我觉得，真正牛逼的人都不需要考虑工具，他们都应该是扫地僧或者香克斯，刀未出鞘就能把周围一圈人干翻。可惜我不牛逼，还有点中二，所以我时常以思考如何选择工具为乐趣。初出茅庐的中二人基本上都有个特点，就是喜欢极力维护自己使用的工具之作为工具的强大还有贬低那些别人使用的早已泛滥成灾的工具为乐趣。依我看用五块钱的皮揣子和一百块钱的皮揣子都能通马桶，不过你硬要说你这个一百块钱的皮揣子被哪位和高尔夫老板一样的管道清洁工附过魔我也会觉得情有可原。

虽然我觉得选择工具十分搞笑，不过如上文所说，我也对我自己使用的工具十分的picky，因为我自己也经常觉得自己好笑，并且我觉得阐述以下我自己的日常休闲作业用的工具体系也没什么坏处，便就打开手里这台被Fedora化的Thinkpad开始敲文章了。

对我来说，电脑永远是Primary Device, 而手机大部分时间连Tertiary Device都排不上，不过愚蠢的现代社会迫使手机成为所有人的Primary Device，我也在努力的反抗，不过似乎成效甚微。不过在我心中，要在我学校这种加州的荒山野岭上生活，少了我那辆离合器已经快要烧没了的986就像少了一条腿，不过那车的火花塞和离合器基本上是以后肯定要换的，所以我现在严格意义上是瘸了一条腿的三足生物；作为一名无时不刻在装作自己有智商的人，我永远不会把手机当作自己的第二大脑，尽管我现在常用的有三台智能手机，不过依我看，不管是安卓还是iOS，里面大部分都是些只会污染你心灵的东西。手机唯一不可被替代的作用就是可以随时随地的通话，而打字而言，不要说写作，就算及时聊天用电脑也比手机高效的多。不过你看那些整天想问题的人有谁会在微信上跟同年级的妹妹在那唧唧我我的，依我看，我父母谈恋爱的时候几天写一次信可不比现在小情侣整天微信上发早安晚安浪漫的多？大部分人类的主要交流只需要讲话或者写字，讲话是为了能够直接具体的交流，而写字很大程度上是为了让人思考，换句话说是为了让东西“过脑子”，按照理想状态，信息时代的讲话应该对应电话，而写字应该对应email，可悲的是大部分人两个都不沾，只会在各大即时聊天平台上你一句我一句在那聊个不停，又字字说的含糊不清。

我也不是说即时聊天不好，他有个好处就是你看到什么好玩的东西，或是觉得什么东西特别傻逼，你可以发给朋友并且得到比email还要及时的回应，这点是讲话和email都不具备的，因为讲话不具备meta-data的特质，而email又缺乏一定的及时性，所以之适合深度的交流。不过这种东西对我来说只适合在最好的朋友之间搞搞，因为我们的大部分笑点都充斥了性别歧视和种族主义，发到什么大群或者类推特平台上肯定要被封号，不过你要是说我们的笑话低俗，我肯定觉得你还没有开悟。

关于email无法及时聊天的特质，我觉得这是很好的，既适合密友之间进行高效的深度交流又适合不是很熟的人之间进行保持一定距离又信息密度极高的交流，并且又有极易整理和保存的特性，自己设立了自己的域名又能很容易的就在嘈杂的网络世界里独善其身。有意义的邮件整理下来就能变成书信集，但是你把个什么群聊天记录整理下来试试？最高效，有逻辑性，易于保留和回忆的方式永远是邮件，因为文章永远是最容易通过文字达到可谓有深度的体裁，而不是什么QQ群里的特效聊天气泡。而最高效的及时通讯永远是电话，它保留了除气场，面部和肢体动作之外最直接和完善的信息。至于打字聊天，它不应该成为一个社会上最主流的东西，它可以作为朋友之间偶尔通讯的工具使用，可以作为客服软件使用，但是如果大部分人现在只会打字聊天，那只能说明他们已经退化到连一封问候朋友的100字邮件都写不出的地步了。

还有个奇怪的东西，在中国它叫QQ空间，或者新浪微博，或者微信朋友圈；在国外它叫instagram或者twitter。它在产生之初就试图极力降低人们的智商，让人们产生不写太多字来表达内容的习惯。而在传播的途径上，这玩意有点像以前的无线电台，几个人在那边聊天，或者一个人对着麦克风瞎说八道个一个小时；如果是FM电台，全国的人都能知道它们在推荐什么潘玮柏新的说唱歌曲；如果是AM电台，那多半是些无线电爱好者的测试，谁说一句话整个在频道的人都知道。而现在有谁在instagram上发自己的屁股，就像之前的主持人在FM电台上说什么自己的臀围有130cm一样，还一字一句的说穿条纹格子紧身裤的时候自己的屁股被勒的多紧，你说这听着其不奇怪？反正我是宁愿听他们在那放《谁偷走了我的麦克风》；而朋友圈里谁去迪士尼玩，谁拍个自拍，谁又拿着自己贴了粉色碳纤维的蔡司镜头去大街上扫街了，都会发出来。反正我是觉得，任何人之间交流，比起谁在干什么，更重要的是知道谁脑子里面有什么。

二.

我刚写完上一篇文章没多久，工具确实不是最重要的东西，最重要的是脑壳里面装着的那团有精神的糊糊，它能手动或者自动的不断迭代，产生许多不可思议的东西。我们现在所享用的所有工具，都是从我们祖先那些脑壳里的糊糊里面产生的。我对工具有很多感受，不涉及任何技术细节我就能就关于工具写出一本书来，但是最根本的，知道什么是好东西就行了，然后去探究里面的思想和技术上的细节，我对许多工具背后的思想略知一二，但是我对它技术上的细节很多都不甚了解。了解了之后就会发现很多东西并不是完美的，但是我觉得我们生活的真实世界本质就是残缺，有一个不残缺的世界在宇宙或精神中的某个地方，它是完美的，但是它是更加矛盾的。妥协了矛盾我们就只能生活在残缺的世界里，所有人的使命都是将残缺的世界补全，感受到自己使命的有那么些人，真正或多或少做到的或许能被列在一本几十或者几百页的书上，真正或多做到的大概一张纸上就能列完了。但是补完了又有什么用，世界永远会被平衡在那个残缺程度的状态，我不知道这个世界在往哪里堆，是变大还是变小，不过我暂时假设它残缺的比例是永恒的。

我能给出的建议是，热爱自己做的东西，并且有充分理由的热爱自己的工具。不一定所有的开发者都要用 Vim 和 Emacs，但是我觉得这些都是至少要学一下的，它们里面有些机制很垃圾，但是这两个是最有哲理并且最为有机的编辑器，Visual Studio 还有 JetBrains 很强大，但是他们显得太人造了。所有音乐制作人至少都要用 Reaper 一段时间，并且能够使用各种 Tracker ，这样的 DAW 能够帮助人理解编曲的本质和编曲的有机性。所有使用计算机的人至少要有把所有操作系统都用一遍的意向，至少有时间和精力要去配置自己的lfs。Windows 和 Mac OS 都不完美，甚至大部分时间都显得很垃圾，但是 Linux 和 BSD 的底层也都是错误百出甚至有时又臭又长的，这些系统有生态，但是比较一下，它们其实还不如 Oberon 要先进，不过说实话我到现在用过体验最好的系统是索尼这几世代基于 BSD 开发的那些用于 PlayStation 的系统。写作者都要试着用 LaTex，它代表了计算机系统里最高层次的排版。网上冲浪者要把所有浏览器内核和排版系统都用一遍，就像冲浪者要找到一块好的冲浪板一样。探索工具是有趣的，不说 Vim 和 Emacs 这种民用级别高层次的工具，即使是 DSA 键帽的 Cherry MX2 或者 Fl Studio 这样稍微有意思一点的工具，我相信已经比你身边那些烂逼的脑糊糊要好玩一万倍了。
